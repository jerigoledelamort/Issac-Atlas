# Scripts

## Назначение

Скрипты автоматизации для разработки и исследования.

## Типы скриптов

| Тип | Назначение |
|-----|------------|
| Build | Автоматизация сборки |
| Test | Автоматизация тестирования |
| Analysis | Анализ данных |
| Utility | Вспомогательные утилиты |

## Структура

```
scripts/
├── build/         # Скрипты сборки
├── test/          # Скрипты тестирования
├── analysis/      # Скрипты анализа
└── utility/       # Вспомогательные скрипты
```

## Языки

- Bash/PowerShell для системных скриптов
- Python для скриптов анализа
- Другие по необходимости

## Использование

```bash
# Пример
./scripts/build/build.sh
./scripts/test/run-tests.sh
```
