# Atlas API

## Назначение

REST API Isaac Atlas.

## Эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/v1/world` | Генерация мира |
| GET | `/api/v1/seed/:seed` | Получение данных о сиде |
| POST | `/api/v1/search` | Поиск сидов |
| GET | `/api/v1/statistics` | Статистика |

## Зависимости

- Isaac Engine
- HTTP сервер
