# Isaac Atlas (Приложение)

## Назначение

**Isaac Atlas** — пользовательское приложение для работы с Isaac Engine.

## Характеристики

- ✅ Использует Isaac Engine
- ❌ **НЕ** содержит логики генерации мира
- ✅ Предоставляет пользовательские интерфейсы

## Модули

| Модуль | Назначение |
|--------|------------|
| `cli/` | Командная строка |
| `gui/` | Графический интерфейс |
| `api/` | REST API |
| `web/` | Веб-интерфейс |
| `assets/` | Ресурсы (изображения, иконки, шрифты) |
| `config/` | Конфигурация приложения |

## Возможности

- Генерация мира по seed'у
- Просмотр сгенерированных миров
- Поиск сидов по параметрам
- Анализ маршрутов
- Поиск экстремальных сидов
- Экспорт данных

## Зависимости

Atlas **зависит** от:
- Isaac Engine

Atlas **не зависит** от:
- Внутренней реализации Engine

## Разработка

Интерфейсы разрабатываются после завершения Engine.
