# Atlas CLI

## Назначение

Командная строка Isaac Atlas.

## Возможности

- Генерация мира по seed'у
- Перебор сидов
- Поиск по параметрам
- Экспорт данных

## Примеры использования

```bash
# Будущие команды
atlas generate --seed 12345
atlas search --items "brimstone"
atlas enumerate --floor basement
```

## Зависимости

- Isaac Engine
