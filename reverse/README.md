# Reverse Engineering

## Назначение

Материалы reverse engineering бинарного файла игры.

## Содержимое

- Аннотированные дампы памяти
- Перечисленные функции
- Структуры данных
- Карта памяти
- Заметки по декомпиляции

## Инструменты

Материалы для работы с:
- Ghidra
- IDA Pro
- Radare2
- Другие инструменты RE

## Правила

1. Все наблюдения документируются
2. Все выводы проверяются
3. Связь с исследованиями в `research/`

## Структура

```
reverse/
├── memory-map/      # Карта памяти
├── functions/       # Перечисленные функции
├── structures/      # Выявленные структуры
├── notes/           # Заметки
└── dumps/           # Дампы (исключены из git)
```
