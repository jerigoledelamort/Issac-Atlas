# Tests

## Назначение

Тесты для Isaac Engine и Isaac Atlas.

## Типы тестов

| Тип | Назначение |
|-----|------------|
| Unit tests | Тестирование отдельных модулей |
| Integration tests | Тестирование взаимодействия модулей |
| Verification tests | Сравнение с оригинальной игрой |
| Regression tests | Предотвращение регрессий |

## Структура

```
tests/
├── unit/           # Unit тесты
├── integration/    # Integration тесты
├── verification/   # Тесты верификации
├── regression/     # Регрессионные тесты
└── data/           # Тестовые данные
```

## Правила

1. Тесты должны быть детерминированными
2. Тесты верификации должны сравнивать с оригиналом
3. Тестовые данные должны быть задокументированы

## Запуск

```bash
# Будущая команда
make test
```
