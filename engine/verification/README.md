# Engine Verification

## Назначение

Верификация совпадения с оригинальной игрой.

## Ответственность

- Сравнение с оригинальной игрой
- Проверка совпадения RNG
- Проверка совпадения генерации
- Генерация отчётов о расхождениях

## Зависимости

- `rng/` — для проверки RNG
- `generation/` — для проверки генерации
- `serialization/` — для сравнения данных
