# Isaac Engine

## Назначение

**Isaac Engine** — библиотека для воспроизведения генерации мира The Binding of Isaac: Repentance.

## Характеристики

- ❌ **НЕ** имеет графического интерфейса
- ❌ **НЕ** зависит от GUI
- ❌ **НЕ** зависит от веб-интерфейса
- ✅ Содержит **только** логику воспроизведения генерации

## Модули

| Модуль | Назначение |
|--------|------------|
| `core/` | Ядро движка, инициализация, жизненный цикл |
| `rng/` | Воспроизведение алгоритма RNG игры |
| `generation/` | Генерация этажей, комнат, предметов |
| `serialization/` | Сериализация и десериализация данных |
| `database/` | Хранение и поиск сгенерированных миров |
| `structures/` | Структуры данных игры |
| `verification/` | Проверка совпадения с оригиналом |
| `utils/` | Вспомогательные утилиты |

## Использование

Engine предназначен для использования как библиотека:

```cpp
// Псевдокод
#include <isaac/engine>

Engine engine;
engine.setSeed(seed);
World world = engine.generate();
```

## Зависимости

Engine **не зависит** от:
- Atlas
- GUI
- Веб-интерфейсов
- Сторонних библиотек (кроме стандартной библиотеки C++)

## Разработка

Перед реализацией любого модуля должно быть завершено соответствующее исследование в `research/`.

## Тестирование

Каждый модуль должен иметь тесты, проверяющие совпадение с оригинальной игрой.
