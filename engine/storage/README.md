# Engine Storage

## Назначение

Хранение и управление сгенерированными мирами в памяти.

## Ответственность

- Хранение сгенерированных миров
- Индексация по параметрам
- Поиск и фильтрация
- Управление кэшем

## Зависимости

- `serialization/` — для сериализации
- `structures/` — для структур данных
