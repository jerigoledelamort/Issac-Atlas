# Examples

## Назначение

Примеры использования Isaac Engine.

## Примеры

| Пример | Описание |
|--------|----------|
| `basic_generation/` | Базовая генерация мира |
| `seed_enumeration/` | Перебор сидов |
| `world_analysis/` | Анализ миров |

## Запуск

```bash
cd examples/basic_generation
cmake ..
./basic_generation
```
