## Типы запросов

### Баги

Создайте issue с:
- Описанием проблемы
- Шагами воспроизведения
- Ожидаемым поведением
- Версией проекта
- Платформой

### Фичи

Создайте issue с:
- Описанием фичи
- Обоснованием
- Примерами использования

### Вопросы

Используйте **GitHub Discussions** для:
- Вопросов по использованию
- Вопросов по архитектуре
- Общих обсуждений

### Безопасность

См. [SECURITY.md](SECURITY.md) для уязвимостей.
