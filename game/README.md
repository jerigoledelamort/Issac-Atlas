# Game

## Назначение

Хранение материалов, связанных с оригинальной игрой The Binding of Isaac: Repentance.

## Содержимое

- Копии исполняемых файлов (для исследования)
- Дампы памяти
- Скриншоты
- Записи геймплея
- Сохранения игры

## Важно

⚠️ **Не коммитьте полные копии игры в репозиторий.**

Используйте `.gitignore` для исключения бинарных файлов игры.
Храните ссылки на файлы или используйте git-lfs при необходимости.

## Использование

Материалы из этой директории используются для:
- Reverse Engineering
- Сравнения с результатами генерации
- Верификации гипотез
