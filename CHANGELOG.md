## Формат записей

#### Добавлено (Added)
Для новых возможностей.

#### Изменено (Changed)
Для изменений в существующей функциональности.

#### Удалено (Deprecated)
Для скоро удаляемых возможностей.

#### Удалено (Removed)
Для удалённых возможностей.

#### Исправлено (Fixed)
Для любых исправлений багов.

#### Безопасность (Security)
В случае уязвимостей.

## Версионирование

Проект использует [Semantic Versioning](https://semver.org/lang/ru/):

- **MAJOR** — несовместимые изменения API
- **MINOR** — новая функциональность с обратной совместимостью
- **PATCH** — исправления багов с обратной совместимостью

До выпуска версии 1.0.0 все изменения считаются нестабильными.
