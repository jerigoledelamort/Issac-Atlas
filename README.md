# Isaac Atlas

## Описание проекта

**Isaac Atlas** — долгосрочная исследовательская платформа для полного воспроизведения генерации мира игры **The Binding of Isaac: Repentance** и последующего построения базы данных всех внутренних сидов.

---

## Главная цель

Создать программный комплекс, состоящий из двух независимых частей:

### 1. Isaac Engine (Библиотека)
- **НЕ имеет графического интерфейса**
- **НЕ зависит от GUI**
- **НЕ зависит от веб-интерфейса**
- Содержит исключительно движок воспроизведения генерации мира

**Возможности Engine:**
- Декодирование сидов
- Воспроизведение RNG
- Генерация мира
- Сериализация результата
- Проверка совпадения с оригинальной игрой
- Использование как библиотеки другими приложениями

### 2. Isaac Atlas (Приложение)
- Пользовательский продукт
- Использует Isaac Engine
- **НЕ содержит логики генерации мира**

**Возможности Atlas:**
- CLI (командная строка)
- GUI (графический интерфейс)
- REST API
- Веб-интерфейс
- Поиск сидов
- Просмотр миров
- Поиск предметов
- Анализ маршрутов
- Поиск экстремальных сидов

## Процесс разработки

### Философия проекта

1. **Сначала исследование** — изучение оригинальной игры
2. **Потом документация** — фиксация обнаруженных закономерностей
3. **Потом реализация** — написание кода на основе документации
4. **Потом тестирование** — проверка совпадения с оригиналом
5. **Только потом перебор сидов** — enumerator и database

### Правила разработки

- Запрещено использовать неподтверждённые гипотезы в коде
- Запрещено писать генератор до завершения исследования
- Запрещено изменять алгоритмы оригинальной игры
- Каждое исследование документируется
- Каждая гипотеза получает статус
- Сначала доказательство, потом код

## Быстрый старт

См. [DEVELOPMENT.md](DEVELOPMENT.md) для подробного руководства.

```bash
# Клонирование репозитория
git clone <repository-url>
cd isaac-atlas

# Сборка
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Debug
cmake --build .

# Тесты
ctest --output-on-failure
```
