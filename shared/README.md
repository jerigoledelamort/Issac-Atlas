# Shared

## Назначение

Общие структуры данных и определения, которые могут использоваться как Isaac Engine, так и Isaac Atlas.

## Содержимое

- Общие определения типов
- Общие форматы данных
- Общие константы
- Протоколы обмена данными

## Правила использования

1. Shared не должен содержать логики
2. Shared не должен зависеть от Engine или Atlas
3. Shared должен быть максимально стабильным

## Примеры

- Форматы сериализации
- Константы игры
- Определения типов данных
