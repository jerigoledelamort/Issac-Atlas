# Tools

## Назначение

Инструменты разработки, не являющиеся частью основного проекта.

## Содержимое

- Локальные утилиты
- Инструменты анализа
- Вспомогательные программы
- Прототипы

## Правила

1. Инструменты не должны зависеть от Engine
2. Инструменты могут использовать Engine как библиотеку
3. Каждый инструмент документируется

## Структура

```
tools/
├── seed-analyzer/   # Анализатор сидов
├── world-compare/   # Сравнение миров
└── data-extractor/  # Извлечение данных
```
