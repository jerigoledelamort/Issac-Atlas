# Research

Хранение исследований, гипотез и их статусов по проекту Isaac Atlas.

> «Сначала исследование, потом документация, потом реализация»

## Структура

```
research/
├── binary-passport.md     # PE-заголовки, хеши, метаданные isaac-ng.exe
├── hashes.md              # Криптографические хеши файлов игры
├── version.md             # Версия игры (v1.9.7.17)
│   ├── rng/                   # Исследование RNG
│   ├── rng-architecture.md    # Архитектура RNG: XORSHIFT, функции, pipeline
│   └── seed-decoding.md       # Спецификация декодера сидов (FUN_009EB6B0)
├── ghidra/                # Проект Ghidra
│   ├── IsaacAtlas_RE.gpr  # Файл проекта
│   └── Decomp/            # Декомпилированные функции
│       └── 009eb6b0.c     # Исходник FUN_009EB6B0 (seed decoder)
└── original/              # Оригинальные файлы игры
    ├── game/              # isaac-ng.exe, DLL
    └── resources/         # packed-архивы (.a), Lua-скрипты
```
