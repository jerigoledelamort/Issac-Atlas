# Research

## Назначение

Хранение исследований, гипотез и их статусов по проекту Isaac Atlas.

## Философия

> «Сначала исследование, потом документация, потом реализация»

Каждое исследование должно быть завершено **до** начала реализации.

---

## Структура каталогов

### Reverse Engineering инфраструктура

| Каталог | Назначение |
|---------|------------|
| `original/` | Оригинальные файлы игры (бинарники, ресурсы) для анализа |
| `ghidra/` | Проекты Ghidra, скрипты декомпиляции |
| `exports/` | Экспортированные данные из Ghidra (структуры, сигнатуры) |
| `pseudocode/` | Псевдокод ключевых функций |
| `callgraphs/` | Графы вызовов функций |
| `notes/` | Заметки исследования, наблюдения, идеи |
| `experiments/` | Эксперименты, тестовые запуски, валидация гипотез |

### Тематические исследования

| Каталог | Назначение |
|---------|------------|
| `bosses/` | Исследование логики боссов |
| `enemies/` | Исследование поведения врагов |
| `floors/` | Исследование генерации этажей |
| `generation/` | Исследование общей генерации мира |
| `items/` | Исследование системы предметов |
| `mechanics/` | Исследование игровых механик |
| `rng/` | Исследование генератора случайных чисел |
| `rooms/` | Исследование генерации комнат |

## Структура исследования

```
research/<тема>/
├── hypothesis.md      # Описание гипотезы
├── evidence.md        # Доказательства
├── tests.md           # Тестовые данные
├── analysis.md        # Анализ
└── status.md          # Статус
```

## Статусы гипотез

| Статус | Описание |
|--------|----------|
| `Новая` | Только предложена |
| `В проверке` | Требуется верификация |
| `Подтверждено` | Проверена и принята |
| `Отвергнуто` | Не подтвердилась |

## Критерии принятия

Гипотеза принимается для использования в коде, когда:
- ✅ Есть эмпирические доказательства
- ✅ Статус = «Подтверждено»
- ✅ Документация завершена
- ✅ Тестовые данные собраны
