## Как вычислить хеши

### Windows (PowerShell)

```powershell
# SHA-256
Get-FileHash -Path "path\to\file.exe" -Algorithm SHA256

# MD5
Get-FileHash -Path "path\to\file.exe" -Algorithm MD5

# Размер файла
(Get-Item "path\to\file.exe").Length
```

### Linux / macOS

```bash
# SHA-256
sha256sum path/to/file.exe

# MD5
md5sum path/to/file.exe

# Размер
stat -c%s path/to/file.exe  # Linux
stat -f%z path/to/file.exe  # macOS
```

## Рекомендуемые файлы для сохранения

Для воспроизводимого исследования необходимо сохранить следующие файлы игры:

### Обязательные

- `isaac-ng.exe` — основной исполняемый файл
- Все `.dll` файлы из директории игры
- `repentance/` — директория с дополнением Repentance

### Данные игры

- `resources/` — архивы ресурсов (`.arc`)
- `data/` — игровые данные (`.xml`, `.txt`)
- `scripts/` — скрипты Lua (если доступны)

### Локализация

- `lang/` или `localization/` — файлы локализации
