## Общая цепочка работы RNG

На текущий момент подтверждена следующая последовательность:

```
┌─────────────────────────────────────────────────────────────────┐
│                    ПОЛЬЗОВАТЕЛЬСКИЙ ВВОД                       │
│                    (String Seed / Seed Menu)                    │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  ┌───────────────────────────────────────────────────────────┐  │
│  │  FUN_009EB6B0 — Seed Decoder                              │  │
│  │  ───────────────────────────────────────────────────────  │  │
│  │  • Ожидает строку 9 символов: "XXXX XXXX"                 │  │
│  │  • Base32-декодирование (алфавит: 32 символа)             │  │
│  │  • XOR с константой 0x0FEF7FFD                            │  │
│  │  • Проверка checksum (9-й символ)                         │  │
│  │  • Возвращает: uint32 Internal Seed                       │  │
│  └───────────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  ┌───────────────────────────────────────────────────────────┐  │
│  │  FUN_009EB880 — RNG Initialization                        │  │
│  │  ───────────────────────────────────────────────────────  │  │
│  │  • Принимает готовый uint32 seed                          │  │
│  │  • Инициализирует структуру RNG                           │  │
│  │  • Устанавливает initialized flag                         │  │
│  │  • Сохраняет текущее значение seed                        │  │
│  │  • Устанавливает три параметра сдвига (shift values)      │  │
│  │  • НЕ создаёт новый сид                                   │  │
│  └───────────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  ┌───────────────────────────────────────────────────────────┐  │
│  │  FUN_006EF890 — RNG Step                                  │  │
│  │  ───────────────────────────────────────────────────────  │  │
│  │  • Проверяет, что RNG инициализирован                     │  │
│  │  • Проверяет, что seed != 0                               │  │
│  │  • Обновляет seed по алгоритму XORSHIFT:                  │  │
│  │      x ^= x >> a;                                         │  │
│  │      x ^= x << b;                                         │  │
│  │      x ^= x >> c;                                         │  │
│  │  • Возвращает следующее псевдослучайное значение          │  │
│  └───────────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                    ИГРОВАЯ ЛОГИКА                               │
│              (генерация этажей, предметов, врагов)              │
└─────────────────────────────────────────────────────────────────┘
```

## Подтверждённые функции

### FUN_009EB6B0 — Seed Decoder

| Характеристика | Значение |
|----------------|----------|
| Назначение | Декодирование строкового сида в uint32 |
| Вход | Строка 9 символов (XXXX XXXX) |
| Выход | uint32 Internal Seed |
| Статус | ✅ Подтверждено |

**Подробности:** [seed-decoding.md](seed-decoding.md)

---

### FUN_009EB880 — RNG Initialization

| Характеристика | Значение |
|----------------|----------|
| Назначение | Инициализация внутреннего состояния RNG |
| Вход | uint32 seed (готовый) |
| Выход | void (инициализирует структуру) |
| Статус | ✅ Подтверждено |

**Структура RNG (минимальная):**

```cpp
struct RNG {
    bool initialized;      // Флаг инициализации
    uint32 seed;           // Текущее значение seed
    uint32 shift_a;        // Параметр сдвига 1
    uint32 shift_b;        // Параметр сдвига 2
    uint32 shift_c;        // Параметр сдвига 3
    // ... возможно другие поля
};
```

**Ключевые наблюдения:**
- Функция **НЕ создаёт новый сид**
- Передается уже готовый `uint32 seed`
- Устанавливает флаг `initialized`
- Сохраняет три параметра сдвига для XORSHIFT

---

### FUN_006EF890 — RNG Step

| Характеристика | Значение |
|----------------|----------|
| Назначение | Вычисление следующего состояния RNG |
| Вход | void (использует внутреннее состояние) |
| Выход | uint32 (следующее псевдослучайное значение) |
| Статус | ✅ Подтверждено |
| Алгоритм | XORSHIFT (Marsaglia) |

**Алгоритм:**

```cpp
uint32 RNG_Step() {
    // Проверки
    if (!initialized) return error;
    if (seed == 0) return error;
    
    // XORSHIFT алгоритм
    x ^= x >> a;    // Правый сдвиг
    x ^= x << b;    // Левый сдвиг
    x ^= x >> c;    // Правый сдвиг
    
    seed = x;
    return x;
}
```

**Ключевые наблюдения:**
- Это **классический XORSHIFT** генератор (George Marsaglia)
- Функция **НЕ генерирует первоначальный сид**
- Только вычисляет следующее состояние генератора
- Требует предварительной инициализации через `FUN_009EB880`

## Связанные функции

### FUN_006F72F0 — Run Preparation

| Характеристика | Значение |
|----------------|----------|
| Назначение | Подготовка игрового состояния для нового забега |
| Статус | ✅ Подтверждено (частично) |

**Наблюдаемое поведение:**
1. Очищает состояние предыдущего ранa
2. Вызывает `FUN_006EF890()` (RNG step)
3. Вызывает `FUN_009EB880(...)` (RNG initialization)

**Неизвестно:**
- Источник первоначального `StartSeed` внутри этой функции
- Точная последовательность вызовов

**Гипотеза:** Используется при начале нового забега или подготовке игрового состояния.

---

|----------------|----------|
| Назначение | Копирование структуры (CopyFrom / operator=) |
| Отношение к RNG | ❌ Практически не имеет |
| Статус | ✅ Подтверждено |

**Наблюдение:** Функция копирования структур данных, не связана с генерацией сидов или RNG.

### FUN_00958CB0 — Seed Forwarder

| Характеристика | Значение |
|----------------|----------|
| Назначение | Передача существующего seed в FUN_009EB880 |
| Отношение к RNG | ⚠️ Косвенное |
| Статус | ✅ Подтверждено (частично) |

**Наблюдение:**
- **НЕ генерирует сид**
- Передает уже существующее значение в `FUN_009EB880`
- Предположительно используется при:
  - Загрузке сохранения
  - Continue
  - Daily Run
  - Challenge
  - Других режимах, где сид уже известен

## Неизвестные элементы

### 🔴 StartSeed Source — Источник первоначального сида

**Статус:** ❌ НЕ НАЙДЕН

**Проблема:**
На данный момент не найдена функция, которая создаёт первый `uint32 StartSeed` при создании совершенно нового забега (New Run без ввода пользовательского сида).

**Возможные источники:**
- Системное время (time() / GetTickCount())
- Аппаратный RNG (если доступен)
- Фиксированное значение + модификаторы
- Комбинация вышеперечисленного

**Текущая задача исследования:**
Именно поиск функции, создающей первый `StartSeed`, является основной задачей исследования.

## Диаграмма состояний RNG

```
┌─────────────┐
│   UNINIT    │ ─────────────────────────────────────┐
│  (start)    │                                       │
└──────┬──────┘                                       │
       │                                              │
       │ FUN_009EB880(seed)                           │
       ▼                                              │
┌─────────────┐                                       │
│  INITIALIZED│ ◄─────────────────────────────────────┘
│  (ready)    │   FUN_006EF890() (step)
└──────┬──────┘   (остается в этом состоянии)
       │
       │ seed == 0 или ошибка
       ▼
┌─────────────┐
│   ERROR     │
│  (invalid)  │
└─────────────┘
```
